- ✅ StateMachineTokenizer (100+ MB/s vs regex 15 MB/s)
- ✅ Longest-match-first strategy
- ✅ Dictionary serialization

## How to Use

### Quick Start - Single Layer

```python
from layer5_optimized import OptimizedLayer5Pipeline

l5 = OptimizedLayer5Pipeline()
compressed = l5.compress(data)
decompressed = l5.decompress(compressed)
```

### Full Pipeline - Maximum Compression

```python
from layer5_optimized import OptimizedLayer5Pipeline
from layer6_optimized import OptimizedLayer6Pipeline
from layer7_optimized import OptimizedLayer7Pipeline

l5 = OptimizedLayer5Pipeline()
l6 = OptimizedLayer6Pipeline()
l7 = OptimizedLayer7Pipeline()

# Compress
step1 = l5.compress(data)
step2 = l6.compress(step1)
step3 = l7.compress(step2)

# Decompress
back1 = l7.decompress(step3)
back2 = l6.decompress(back1)
back3 = l5.decompress(back2)
```
